Display moving smile image on canvas

Description : Load smile image and display it on canvas and pass x and y coordinates of your nose to the image() function so that it moves along with your lips

  1. Load the "smile.jpg" image using loadImage() function of p5.js and store it in smile_img variable (we already have uploaded image "smile.jpg"). Learn more about - loadImage()
  2. Use image() function of p5.js to display the smile image on canvas. Put this code inside draw() function.Learn more about - image()
    1. image function takes 5 parameters loadImage(image, x co-ordiate, y co-ordinate, width, height)
    2. In place of image pass smile_img variable as it holds the lips image.
    3. Then we will use x coordinate of the nose(which is in noseX variable) and pass it in the place of x coordinate of image() function
    4. Then we will use y coordinate of the nose, but we want lips co-ordinate and lips are below nose so add 15 to y co-ordinates of nose(which is in noseY variable therefore noseY+15) and pass it in the place of y coordinate of image() function
    5. And then give any value for the width and height of image() function
  3. write the above code in sketch.js
